Welcome

HELLO and welcome Monday Night Games! We are a discord gaming server with tons of lovely people that play just about any game you do!

This server was created in 2018 and has grown exponentially since then and because of this we have had many growing pains so detailed rules were established.

  • The Rules page will outline the behavior that is expected in the server and is what you should read first

  • We have many channels so you will need to readf that next (yes we have a lot but it helps in keeping multiple conversations from colliding)

  • The FAQ page is very helpful and is a must-read so make sure to check there

  • Lastly we have the information page which will give more specific information about our gaming channels (like Destiny 2 for example) that will complement getting involved in a community

Most of us don't bite so don't be shy and get involved! Make sure you read the rules and guides before you go otherwise, enjoy!
